AutomaticCRR

AutomaticCRR is a command line
software running under linux its
goal is to simplify the cracking of
wifi networks.

Step 1 : Search for linux commands.

First of all I searched on the internet for linux commands to crack a wifi network,
I found this site (FR). Then I made a list of all the commands.

Step 2 : Create an AutomaticCRR file.

First of all we have to find the location of the script and then check
if the AutomaticCRR folder doesn't exist, if it does we create it
otherwise we don't do anything.

1 | import os
2 | PATH = os.getcwd() #We get the path where the program is
3 | PATH = (f"{PATH}/")
4 | if not os.path.exists(f"{PATH}AutomaticCRR"): #We check that the file exists
5 |     os.makedirs(f"{PATH}AutomaticCRR") #We create the file
6 | PATH = (f"{PATH}AutomaticCRR/")
7 | os.system(f"cd {PATH}") #We go to the folder with the command "cd"
8 | os.system(f"sudo rm -R {PATH}*") #Delete everything in the file

How to launch it ?

You can launch it by typing the command
"python3 AutomaticCRR.py" in terminal and
by being in the folder where the py file is
and by being in super user mode.
Or run a shell script in super user mode that will
do everything for you :

1 | #!/bin/bash
2 | #Please do the command "chmod +x Launch.sh" before launch the script
3 | if test $(whoami) = 'root' ; then
4 |    python3 AutomaticCRR.py
5 | else
6 |    echo Please run the script as a super user
7 |    exit
8 | fi
